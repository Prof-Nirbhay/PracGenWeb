How to Use the Website

1

Select Class and Semester

2

Enter the Subject Name

3

Add Main Questions (one per line)

  • Write only one main question per line.
  • Press Enter after each main question.
  • Each new line is treated as a new main question.
  • Do not combine multiple main questions in one line.
4

Add Sub-Questions using TAB

  • Press TAB before typing a sub-question.
  • Sub-questions will appear as (a), (b), (c).
  • Sub-questions belong to the previous main question.
  • Do not press Enter without TAB if it is a sub-question.
5

Click Generate to preview or download the paper
